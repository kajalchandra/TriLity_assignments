
# TriLity Assignment 1 React + TypeScript

## project Description

The project has

- Reusable React Components
- TypeScript interfaces for component props
- React router for navigation
- organized folder structure

## tech stack
- React
- TypeScript
- Vite
- React router dom
- CSS


## Setup

## Project Creation

The project was created using Vite with the following command:

npm @


Project configuration:

- Framework: React
- Variant: TypeScript

## Setup
run on terminal -  npm create vite@latest

Configure project 

-framework - React
-Variant - TypeScript

## install project dependencies
- for Routing run  
 npm install react-router-dom

## Run the development server run
  npm run dev

## Build the Project
 npm run build
